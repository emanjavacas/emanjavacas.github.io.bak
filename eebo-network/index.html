Click on a node to display associated info

General info

  • The area of each node represents the vertex degree of that node (i.e number of adjacent edges). In order to make it more visible, size is actually mapped to the log of the degree instead of degree itself.
  • Color is mapped to the corresponding generation. Currently no legend is provided. But it can be easily found out by clicking on the nodes an opening the Author Info panel.
  • The distribution of adjacent nodes by generation is displayed in the histogram at the bottom of the Author Info panel
